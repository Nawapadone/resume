# Theme B Redesign — Dark Charcoal Emerald

**Date:** 2026-06-26  
**Status:** Approved

## Problem

Theme B background `#08080f` is near-black (almost pure black), making the resume feel too heavy and hard to read. The user wants a lighter dark theme that stays visually distinct from Theme C (warm cream `#faf8f5` + amber).

## Decision

Replace the indigo/purple palette of Theme B with **Dark Charcoal Emerald**:
- Background: `#141414` (charcoal, ~40% lighter than current)
- Accent: emerald `#34d399` / `#6ee7b7`

## Constraints

- **Theme C must not be touched.** Every change is scoped to `.dark` selectors or `html.dark` selectors only.
- No layout, spacing, font, or structural changes.
- No changes to `:root:not(.dark)`, global `body`, or any Tone C specific selectors.
